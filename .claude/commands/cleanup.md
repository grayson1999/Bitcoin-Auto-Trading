# 코드 정리 및 검증

**대상: 현재 작업 파일만**

## 코드 정리
- Unused imports/variables 제거
- 중복 코드 함수/유틸로 추출 (기존 유틸 활용)
- Magic number/string 상수화
- 명시적 타입, 일관된 네이밍
- 불필요한 주석 제거

## 검증
1. **계획 준수** - plan.md, tasks.md, CLAUDE.md 대비 구현 일치
2. **논리** - 조건문 오류, 엣지케이스, 비동기, 에러핸들링, 리소스 누수
3. **보안** - injection, secrets, 입력 검증
4. **최적화** - N+1, 캐싱, memo, lazy loading

## 출력
```
## 계획: [O/X] 항목
## 이슈: [심각도] 파일:라인 - 설명
## 제안: 변경사항
```

실행 전 확인 필수
